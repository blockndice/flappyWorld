paramètre      │ avant  │ après  │ rôle
───────────────┼────────┼────────┼──────────────────────────────────────────
FLAP_VY        │  -7.2  │  -7.6  │ élan au saut (plus négatif = plus fort)
FLOAT_FRAMES   │   —    │   14   │ frames de gravity réduite (maintien btn)
FLOAT_GRAVITY  │   —    │  0.10  │ gravity pendant le float (normal = 0.38)
GRAVITY        │  0.38  │  0.38  │ inchangé
PIPE_SPEED     │   2.4  │   2.4  │ inchangé
PIPE_GAP       │  130   │  130   │ inchangé

conseils de réglage :
float trop long     → baisser FLOAT_FRAMES   (ex: 10)
vol trop léger      → monter  FLOAT_GRAVITY   (ex: 0.18)
saut trop puissant  → remonter FLAP_VY        (ex: -7.3)
