boucle à pas fixe (v0.17.2)
────────────────────────────────────────────────────────────────────────────
  problème : requestAnimationFrame cadencé sur le Hz écran
             → 120Hz = jeu 2× plus rapide qu'un 60Hz

  solution : accumulateur à pas fixe (STEP = 1000/60 = 16.67ms)
             la physique avance toujours 60 fois/seconde réel
             le rendu suit la cadence native (plus fluide sur 120Hz+)

  dans le pas fixe (60/s)  │ dans render() (cadence native)
  ─────────────────────────┼──────────────────────────────
  physique oiseau           │ dessin pur
  tuyaux (spawn + move)     │ shake mort (cosmétique)
  sol, nuages               │ UI HTML (volWrap)
  tous les compteurs        │
  particules                │
  coinTick                  │

  garde-fou : cap 50ms → pas de saut si l'onglet reprend du focus
